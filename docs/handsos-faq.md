## 1. 영업시간 vs 직원 근무시간 (가장 많이 헷갈리는 부분!)

### Q. 영업시간과 직원 근무시간은 뭐가 다른가요?

| 구분 | 영업시간 | 직원 근무시간 |
|------|----------|---------------|
| **의미** | 매장이 문을 여는 시간 | 각 직원이 실제로 일하는 시간 |
| **설정 위치** | 매장관리 > 매장정보 (또는 기본설정) | 매장관리 > 직원관리 > 근무스케줄 |
| **적용 대상** | 매장 전체 (1개) | 직원 개인별 (각각 다름) |
| **예약 영향** | 영업시간 밖의 시간에는 예약 자체가 안 잡힘 | 해당 직원이 근무하지 않는 시간에는 그 직원에게 예약 불가 |
| **예시** | 매장 10:00~21:00 | A디자이너 10:00~19:00, B디자이너 12:00~21:00 |

### Q. 영업시간을 10:00~21:00으로 설정했는데, 왜 특정 직원한테 9시 예약이 안 잡히나요?

**영업시간이 예약 가능 범위의 "최대 틀"**이기 때문입니다.
- 영업시간: 10:00~21:00 → 이 범위 밖(9시, 22시 등)은 아예 예약 불가
- 직원 근무시간은 영업시간 **안에서만** 설정 가능
- 즉, 매장 영업시간이 10시부터인데 직원을 9시부터로 설정할 수 없음

### Q. 직원마다 근무시간이 다른데 어떻게 설정하나요?

1. **매장관리** > **직원관리** 메뉴 진입
2. 해당 직원 선택
3. **근무스케줄** (또는 근무시간) 탭에서 요일별 시작/종료 시간 설정
4. 휴무일은 해당 요일을 "휴무"로 체크

> **팁**: 직원의 근무시간을 설정해야 예약 시 해당 직원이 가능한 시간대만 노출됩니다.
> 설정 안 하면 영업시간 전체가 예약 가능으로 뜨므로, 없는 시간에 예약이 잡힐 수 있습니다!

### Q. 영업시간만 설정하고 직원 근무시간을 안 넣으면 어떻게 되나요?

직원별 근무시간을 설정하지 않으면, 기본적으로 **영업시간 전체가 해당 직원의 근무시간**으로 간주됩니다.
- 결과: 쉬는 날에도 예약이 잡힐 수 있음
- **반드시 직원별 근무시간과 휴무일을 개별 설정하세요!**

## 2. 기본 설정

### Q. 매장 기본정보는 어디서 수정하나요?

**매장관리** > **매장정보** (또는 기본설정)에서 다음 항목을 설정합니다:
- 매장명, 전화번호, 주소
- 영업시간 (시작~종료)
- 정기 휴무일 (예: 매주 월요일)
- 임시 휴무 설정

### Q. 정기 휴무일과 임시 휴무는 어떻게 다른가요?

- **정기 휴무**: 매주 반복되는 휴무 (예: 매주 월요일) → 기본설정에서 요일 체크
- **임시 휴무**: 특정 날짜만 쉬는 경우 (예: 명절, 인테리어 공사) → 매장관리 > 일정관리에서 해당 날짜 설정

## 3. 예약 관리

### Q. 예약을 잡는 기본 순서는?

1. **예약관리** 메뉴 진입
2. 원하는 날짜 선택
3. 시간대 선택
4. 고객 선택 (기존 고객 검색 또는 신규 등록)
5. 시술 메뉴 선택
6. 담당 디자이너 선택
7. 저장

### Q. 예약 시간 간격은 어떻게 조절하나요?

**설정** > **예약설정**에서 예약 시간 단위를 변경할 수 있습니다.
- 보통 30분 또는 1시간 단위
- 시술 소요시간에 맞게 설정하는 것이 좋음

### Q. 예약이 겹치면 어떻게 되나요?

핸드SOS는 같은 직원에게 같은 시간에 중복 예약을 허용하지 않습니다.
- 다른 직원에게는 같은 시간에 예약 가능
- 예약 충돌 시 알림이 표시됨

## 4. 직원 관리

### Q. 직원 계정(권한)은 어떻게 설정하나요?

**설정** > **시스템 권한설정**에서:
1. 직원 계정 등급 선택 (원장/매니저/디자이너/인턴 등)
2. 좌측 메뉴별 **사용권한** 버튼 클릭
3. 각 메뉴에 대한 접근 권한 설정 (열람/수정/삭제)

> **주의**: 직원 계정에서 출퇴근 현황 수정/삭제를 막으려면 매장관리 메뉴의 권한을 "열람만 가능"으로 설정하세요.

### Q. 직원 출퇴근은 어떻게 기록하나요?

- 직원이 자신의 계정으로 로그인 후 출근/퇴근 버튼 클릭
- 원장 계정에서 **매장관리** > **근태현황**에서 전체 직원의 출퇴근 내역 확인 가능
- 근태 항목: 출근, 퇴근, 지각, 조퇴, 휴무, 당직, 외근

### Q. 직원이 퇴사하면 어떻게 처리하나요?

**매장관리** > **직원관리**에서 해당 직원을 "비활성" 처리합니다.
- 삭제하면 해당 직원의 과거 매출/예약 기록도 사라질 수 있으므로 비활성 처리 권장
- 비활성 직원은 예약 배정에서 제외됨

## 5. 매출 / 정산

### Q. 일일 매출은 어디서 확인하나요?

**매출관리** > **일일매출** (또는 매출현황)에서 확인 가능합니다.
- 날짜별, 직원별, 시술 메뉴별 매출 조회
- 카드/현금/기타 결제수단별 구분

### Q. 직원 급여(커미션) 자동 계산이 되나요?

핸드SOS는 매출에 따른 **급여 자동 계산** 기능을 제공합니다.
- **매장관리** > **급여관리**에서 직원별 급여 체계 설정
- 기본급 + 인센티브(커미션) 비율 설정 가능
- 월별 정산 내역 자동 생성

## 6. 일정 / 스케줄

### Q. 매장 일정(이벤트)은 어떻게 등록하나요?

1. **매장관리** > **일정관리** 메뉴 진입
2. 달력에서 원하는 날짜의 **+ 버튼** 클릭
3. 스케줄 제목과 세부 내용 입력
4. 저장

> 등록된 일정은 달력에 표시되어 직원들이 공유할 수 있습니다.

### Q. 매장 공지사항은 어떻게 올리나요?

1. **매장관리** > **공지사항** 메뉴 진입
2. 등록 버튼 클릭
3. 공지 내용 작성 후 등록

## 7. 모바일 앱

### Q. 모바일에서도 사용할 수 있나요?

네, 핸드SOS는 모바일 앱을 제공합니다.
- **iOS**: App Store에서 "HandSOS" 검색
- **Android**: Google Play에서 "핸드SOS" 검색
- PC와 동일한 계정으로 로그인하면 데이터 동기화

### Q. 모바일 매장 홈페이지란?

핸드SOS에서 제공하는 **모바일 전용 매장 소개 페이지**입니다.
- 매장 정보, 메뉴, 가격, 직원 소개 등을 고객에게 보여줄 수 있음
- 별도 홈페이지 제작 없이 핸드SOS 내에서 자동 생성
- 네이버 연동도 가능

## 8. 자주 하는 실수 & 꿀팁

### 실수 모음

| 실수 | 결과 | 해결 |
|------|------|------|
| 영업시간만 설정하고 직원 근무시간 미설정 | 쉬는 직원에게도 예약 잡힘 | 직원별 근무스케줄 반드시 개별 설정 |
| 정기 휴무일 미설정 | 쉬는 날에 예약/알림 발생 | 매장정보에서 정기 휴무 요일 체크 |
| 퇴사 직원 삭제 | 과거 매출 기록 손실 | 삭제 대신 "비활성" 처리 |
| 권한 설정 안 함 | 직원이 매출/급여 등 민감 정보 열람 가능 | 시스템 권한설정에서 등급별 메뉴 접근 제한 |

### 꿀팁

1. **매주 월요일에 해당 주 직원 스케줄 확인** → 변경사항 있으면 바로 반영
2. **신규 직원 등록 시 근무시간 + 권한 동시에 설정** → 나중에 잊어버리기 쉬움
3. **임시 휴무는 최소 하루 전에 등록** → 예약 알림이 자동으로 차단됨
4. **매출 입력은 당일에 바로** → 나중에 몰아서 하면 실수 발생 가능
5. **모바일 앱 설치 후 푸시 알림 활성화** → 예약 변경/취소 실시간 알림
